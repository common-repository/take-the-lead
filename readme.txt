Multistep lead generating form. Simple for your visitors and easy to manage

== Description ==

Can be added to any WordPress site where there is a need to capture leads. Works straight out the box - no fiddly setting up needed

Bespoke versions available for Infusionsoft and Hubspot CRM

= Features =

*	Any number of steps
*	You choose what goes in what step
*	Homepage option with background image
*	Simple styling options

== Installation ==

1.	Login to your wordpress dashboard.
2.	Go to 'Plugins', 'Add New' then search for 'Take the Lead'.
3.	Follow the on screen instructions.
4.	Activate the plugin.
5.	To use the plugin in your posts and pages add the shortcode `[takethelead]`.
6.	Change the options in the plugin settings page.
